# Shopping List

## Project Description
This is a shopping list project developed as part of the Rocketseat challenge. The aim of this project is to provide a simple interface to add and delete items from a shopping list. Although it has been completed, the project still contains a few bugs that need to be fixed.

## Features
- **Add Items**: Allows users to add new items to the shopping list.
- **Delete Items**: Allows users to remove existing items from the shopping list.

## Technologies Used
- HTML
- CSS
- JavaScript
